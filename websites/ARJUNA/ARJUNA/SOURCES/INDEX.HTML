Arjuna System Source

Arjuna has been designed to be highly portable and currently runs on most Unix platforms (a port to Windows-NT is underway) and have been ported and tested on the following architectures/compiles:

  • Solaris 2.5.X: cfront 3.0.1, g++ 2.7.2 and SC3.0.1.
  • SunOS 4.1.X: cfront 3.0.1 and g++ 2.7.2.
  • HP-UX 10.X: HP-C++ (A.10.X).
  • Linux 1.2.X: g++ 2.7.2.
  • Linux 2.0.X: g++ 2.7.2.

The following version of the Arjuna System are available:

  • Arjuna 3.3.2 (tar.gz)
