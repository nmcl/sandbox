ArjunaII System

Introduction

ArjunaII is an object-oriented programming system that provides a set of tools for the construction of fault-tolerant distributed applications. The system is constructed using strong separation of interface for implementations. This allows the interface to services within ArjunaII to have multiple implementations.

ArjunaII is constructed out of a set of modules. Each module providing interface to services and/or a set of implementations for those services.

Some of the module within ArjunaII are listed below:

  • Gandiva provides support the construction of software that exploits the strong seperation of interface form implementation. It also provides support for the module construction of a software.
  • Dharma provides support for construction of application which contain "Atomic Object" and "Atomic Actions" (transactions).
