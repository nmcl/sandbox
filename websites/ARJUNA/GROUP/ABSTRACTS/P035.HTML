A Stub Generatoin System For C++

Graham D. Parrington

Abstract: This paper describes the implementation of a Stub Generation system targeted specifically at C++. It enables distribured C++ applications to be constructed in a straightforward manner with minimal programmer assistance. The sysytem does not require the use of an auxiliary interface definition lannguage but instead processes existing C++ header files to maximise transparency. The generated code exploits the power of C++ by using operations overloading for parameters marshalling and constructors and destructors to drive the remote binding process.
