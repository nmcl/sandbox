Arjuna Group Summary

The Arjuna group has spent several years researching into the development of concepts and techniques for building robust distributed systems. This has led to the development of a collection of programming systems, in particular, a system called Arjuna that provides a set of tools for constructing fault-tolerant distributed applications using atomic transactions.

Current work is emphasizing reliability and fault tolerance issues in electronic commerce, network services and Web-based applications.
