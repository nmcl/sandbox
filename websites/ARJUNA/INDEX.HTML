Arjuna Group's Web Site

Welcome to the Arjuna group's web site, part of the Distributed Systems group in the Department of Computing Science of University of Newcastle upon Tyne, England.

The Arjuna group is involved in the construction of tools to allow the development of distributed applications.

Software Systems

As a part of the research undertaken by the Arjuna group a number of software systems have been implemented:

  • Arjuna system is an object-oriented programming system that provides a set of tools for the construction of fault-tolerant distributed applications.
  • W3Objects research is focused on infrastructure support for advanced web-based applications.
  • Shadows is a distributed programming paradigm and a class library which supports that paradigm for C++.
  • ArjunaII is an evolution of the Arjuna system which is currently under development.
  • C++SIM is an object-oriented simulation package written in C++. It provides discrete event process-based simulation similar to SIMULA's simulation class and libraries.
