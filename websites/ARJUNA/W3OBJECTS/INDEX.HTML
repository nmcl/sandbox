W3Objects

Introduction

As part of the Arjuna Project, we are investigating the application of object-oriented technology to the area of the World-Wide Web. We believe that the current web model suffers from poor extensibility characteristics and that an object-oriented architecture would yield a more flexible system. Based on this new architecure we aim to address the problems facing the current web in its primary role as a documentation server and also show how more complex resources and services can be integrated.
