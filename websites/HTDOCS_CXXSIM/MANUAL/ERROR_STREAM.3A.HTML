error_stream - object which provides for output of differ-
       ent levels of error severity messages

SYNOPSIS

       #ifndef ERROR_H_
       #   include <Common/Error.h>
       #endif

       enum ErrorSeverity
       {
            FATAL, WARNING
       };

       extern ostream& operator<< ( ostream& strm,
                                enum ErrorSeverity es );

       #define error_stream cerr

DESCRIPTION

       The error_stream and its associated << operator provide  a
       means  of  outputting  error  messages  which indicate the
       level of severity of the error and give other  information
       which may prove useful in tracking down faults (e.g., pro-
       cess identifier).

       The format for using error_stream is straightforward,  and
       is illustrated with two examples:

       error_stream << WARNING << "Attempt to do something
                 illegal but not severe." << endl;

       error_stream << FATAL << "Attempt to do something
                 such as divide by zero." << endl;
