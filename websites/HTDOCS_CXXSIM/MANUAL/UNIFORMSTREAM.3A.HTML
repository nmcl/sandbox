UniformStream,   Draw,   ExponentialStream,  ErlangStream,
       HyperExponentialStream, NormalStream - various pseudo-ran-
       dom number generators.

SYNOPSIS

       #ifndef RANDOM_H_
       #   include <ClassLib/Random.h>
       #endif

       class Draw
       {
       public:
            Draw (double p, int StreamSelect = 0);
       };

       class UniformStream : public RandomStream
       {
       public:
            UniformStream (double lo, double hi,
                           int StreamSelect = 0);
       };

       class ExponentialStream : public RandomStream
       {
       public:
            ExponentialStream (double Mean,
                               int StreamSelect = 0);
       };

       class ErlangStream : public RandomStream
       {
       public:
            ErlangStream (double Mean, double StdDev,
                          int StreamSelect = 0);
       };

       class HyperExponentialStream : public RandomStream
       {
       public:
            HyperExponentialStream (double Mean, double StdDev,
                                    int StreamSelect = 0);
       };

       class NormalStream : public RandomStream
       {
       public:
            NormalStream (double Mean, double StdDev,
                          int StreamSelect = 0);
       };

DESCRIPTION

       These  classes  provide  pseudo-random  number  generators
       which,  if  given  an integer value n can be used to "dis-
       card" the first n random numbers taken from the base algo-
       rithm.

       With  the  exception of Draw they all have a single public
       method operator() which returns a random number drawn from
       the relevant distribution.

       Draw  returns TRUE with the probability given in the first
       parameter.

       UniformStream returns a number from a uniform distribution
       with the lower bound of lo and an upper bound of hi inclu-
       sive.

       ExponentialStream returns a  number  from  an  exponential
       distribution with Mean as the mean.

       ErlangStream  returns a number from an Erlang distribution
       with a mean of Mean and StdDev as its standard  deviation.

       HyperExponential  returns a number from a hyperexponential
       distribution with a mean of Mean and StdDev as  its  stan-
       dard deviation.

       Normal  returns a number from a normal distribution with a
       mean of Mean and StdDev as its standard deviation.
