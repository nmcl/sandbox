JavaSim Home Page

Welcome to the new JavaSim home page, Department of Computing Science, University of Newcastle upon Tyne, Newcastle upon Tyne, UK.

JavaSim is a set of Java packages for building discrete event process-based simulation, similar to that in Simula and C++SIM (from which JavaSim is derived).

This homepage is now resident on a new machine javasim.ncl.ac.uk

This pages is still under development, so there isn't a lot here at the moment. We hope to remedy that as soon as possible.
